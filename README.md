# Korea-Investment-program
실행 방법 (VS Code / Codespaces)

패키지 설치
터미널에서 아래 명령어를 입력하여 필요한 라이브러리를 설치합니다.

pip install -r requirements.txt

환경변수 설정
Codespaces의 Secret 또는 터미널 환경변수로 아래 3가지를 반드시 설정해야 한다

GH_ACCOUNT : 10자리 계좌번호 (예: 1234567801)

GH_APPKEY : 모의투자 발급 앱 키

GH_APPSECRET : 모의투자 발급 앱 시크릿

프로그램 실행

python main.py
